🚨 의심 판정 알림

서버가 자동으로 판정한 보류/거부 세션 (game_sessions · 최근 50건). 읽기 전용입니다. 새(안 본) 의심만 표시하고, "모두 확인"을 누르면 상단 알림이 사라집니다.

🏅 고득점 세션 (4만+)

4만점 이상 세션 최근 30건 — 자동 판정과 무관한 열람용입니다. 줄을 누르면 플레이 스타일 상세와 요약이 펼쳐져요. (4만 초과 자체는 핵 근거가 아니며, 정상 반영된 기록도 사후 검토용으로 함께 표시)

🛟 점수 누락 복구

오늘 실제로 플레이했지만 랭킹에 반영되지 않은 후보를 찾습니다. 각 유저의 오늘 실제 점수(전체 최고점이 아님)를 기준으로, 현재보다 높을 때만 갱신하고 5만점 이상은 복구하지 않습니다. 자동 복구는 없고 버튼을 눌러야만 실행됩니다.

🔑 핀번호 재설정

닉네임을 잃어버렸거나 다른 기기에서 "계정 연결"이 안 되는 유저를 위해 새 PIN을 발급합니다. 기존 PIN은 조회할 수 없고(서버가 해시만 저장), 완전히 새 PIN으로 교체됩니다. 발급된 PIN은 본인에게 직접 전달해주세요.
직접입력 칸을 비워두면 무작위 4자리가 자동 발급됩니다.

⭐ 리뷰 관리

게임 리뷰(별점+한마디)를 관리합니다. 🔴 새 리뷰 / ✏️ 수정됨 배지가 붙은 항목만 확인하면 됩니다(확인 누르면 배지 꺼짐). 필터에 걸려 보류된 리뷰는 승인해야 공개됩니다. 답글은 게임에 "😺 오잉냥이"로 표시됩니다.

🕵️ 의심 기록 탐지 (수동 스캔)

상위 랭킹 100명을 기준으로 비정상 점수(플레이 대비 과도한 점수, 주간>전체 불일치 등)를 검사합니다. 버튼을 눌렀을 때만 조회합니다.

🗑️ 랭킹 기록 삭제

전체 랭킹 + 이번주 랭킹에서 해당 닉네임 점수를 삭제합니다. (유저 계정/스킨은 유지)

🚫 랭킹 영구차단 (섀도우밴)

이 계정은 게임은 정상으로 되지만 점수가 랭킹에 영구히 안 올라갑니다. 본인 화면엔 "등록 완료"가 그대로 떠서(섀도우밴) 차단 사실을 눈치채기 어렵습니다. 닉네임으로 차단하면 해당 계정(UID) 기준으로 걸립니다. (시크릿모드/새 기기로 새 계정을 파면 빠져나갈 수 있으나, 그 경우 버스트 감지가 다시 잡습니다. 기존에 올라간 점수는 여기서 지우지 않으니 필요하면 위 "랭킹 기록 삭제"를 쓰세요.)

💾 백업

JSON 파일로 다운로드됩니다. 초기화 전 자동 백업과 별개로 수동 백업할 때 사용하세요.
